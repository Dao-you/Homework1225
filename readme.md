# kotlin 程式碼練習

## 心得

隨著實作的東西越來越複雜，甚至涉及到跨 app 的資料讀取，用 Kotlin 寫 Android，比啟用 Java，整體感覺比較簡潔很多，像是語法變少、程式可讀性也比較好。實際寫的時候會發現，用 Kotlin 在處理畫面更新或背景執行時，程式結構比較清楚，也比較不容易寫出一堆冗長的流程。

在實作過程中，也更能體會 View 使用方式對效能的影響，資料一多時，如果沒有好好處理，畫面很容易變卡。另外在背景執行的部分，搭配 Kotlin 的寫法，比起 Java 傳統的方式更直覺，也比較好維護。
